Student Project

Created with AI Assistance

Aurora Software is an innovative project developed by a high school IT student, showcasing the potential of human-AI collaboration. Every line of code was crafted through interaction with Large Language Models, demonstrating how modern AI tools can empower young developers to create professional-grade software.

This project serves as both a functional mobile apps and an educational journey into software development, UI/UX design, and the practical applications of artificial intelligence in programming.

AI Assisted
Open Source
Student Project

The entire development process, from initial concept to final implementation, was guided by AI language models. This includes:

• Core functionality programming
• Performance optimization
• Bug fixing and testing

This transparent approach aims to inspire other students and demonstrate how AI can be leveraged as a powerful educational tool in software development.
